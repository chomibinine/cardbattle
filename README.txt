# Card Battle WebApp - Full Lobby Integrated Version

✅ 통합 기능
- 로비 화면 (index.html)에서 게임 모드 선택 가능
- 닉네임 입력 후 AI 배틀 시작
- 전투 로그 및 승패 요약 기능 포함
- 카드 이미지, 사운드 효과, 덱 시스템 포함

🗂 구조
- index.html: 로비
- ai.html: AI와의 카드 배틀 화면
- js/: 모든 기능 JS 파일
- css/: UI 스타일

📦 향후 확장 가능
- 친구와 온라인 대전 연결
- Firebase 실시간 데이터 연동
- 리플레이, 랭킹 시스템
